# Popular Applications

The [Nix package collection](https://search.nixos.org/packages) contains over 120,000 packages.
Many of them may already work out of the box on Asterinas NixOS,
but it is impractical to test every package
and document its support status.
Instead, we group popular applications into categories
and document the ones we have verified to work.

## Categorization Overview

The categorization is designed to scale to hundreds of applications while serving both server and desktop use cases.
It follows a two-level hierarchy (Category > Sub-category) with the following top-level categories:

| Category | Description |
|----------|-------------|
| [System Core](system-core.md) | Shells, init systems, system monitoring, and essential utilities |
| [Package Management & Containerization](package-management-containerization.md) | Nix package management and container runtimes |
| [Networking](networking.md) | Network utilities, DNS, VPN, and firewalls |
| [Web Servers & Proxies](web-servers-proxies.md) | Web servers and reverse proxies |
| [Databases & Middleware](databases-middleware.md) | Relational databases, NoSQL, search engines, and message queues |
| [Development Tools](development-tools.md) | Language runtimes, build tools, editors, and debugging tools |
| [CI/CD & DevOps](cicd-devops.md) | CI/CD runners and infrastructure automation |
| [Monitoring & Observability](monitoring-observability.md) | Metrics, logging, and tracing tools |
| [Desktop Environments & Display](desktop-environments-display.md) | Desktop environments, window managers, and display servers |
| [Web Browsers](web-browsers.md) | Web browsers |
| [Office & Productivity](office-productivity.md) | Office suites, document viewers, and note-taking apps |
| [Multimedia](multimedia.md) | Video/audio players, graphics tools, and streaming software |
| [Communication](communication.md) | Email clients, instant messaging, and video conferencing |
| [File Management & Terminal Productivity](file-management-terminal.md) | File managers, terminal emulators, archives, and CLI utilities |
| [AI & Machine Learning](ai-machine-learning.md) | Deep learning frameworks, LLM tools, and inference engines |

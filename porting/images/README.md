# Generated Images

This directory is a staging area for binary images produced during the port,
such as:

- `aster-nix.booti` — U-Boot `booti` image generated by `mkimage.py`
- SD card or initramfs images
- Raw memory dumps

Everything in this directory is gitignored by default. Commit generated images
only if they are small, stable, and genuinely useful for reproduction; in that
case add an explicit exception in `.gitignore` or store them elsewhere.
